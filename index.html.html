➕ Comment ajouter une ressource

Ouvrez ce fichier dans le Bloc-notes. Trouvez la section DONNÉES en haut du script. Ajoutez vos fichiers dans la matière qui correspond. Placez les fichiers HTML/DOCX dans le même dossier que ce fichier.

{ titre: "Mon cours sur le handicap",
  type: "cours",
  fichier: "cours_handicap.html",
  desc: "5 séquences avec quiz" }

Types disponibles : cours (HTML) · livret (DOCX) · eval (DOCX) · lien (URL externe)

🌐 Mettre en ligne sur GitHub Pages (gratuit)

  1. Créer un compte sur github.com
  2. Cliquer + New repository → nommer epv-cours → cocher "Public" → Create
  3. Cliquer Add file → Upload files → glisser TOUS les fichiers (HTML + DOCX)
  4. Cliquer Commit changes
  5. Aller dans Settings → Pages → Branch: main → Save
  6. Votre lien : votre-pseudo.github.io/epv-cours → à partager dans Classroom !
